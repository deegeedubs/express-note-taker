# Express Note Taker

## Description

Take notes and save them on this web application. Also, delete notes when they are no longer needed.


## Technology Used <a id = "tech"></a>

- Javascript
- Node.js
- Express.js
- HTML
- CSS
- Heroku

## Summary 

- Express.js is used to handle routing and fetches
- GET, POST, and DELETE methods are used to handle our notes database
- When a new note is entered, a save button appears that will post it to the database and render it on the page
- Each rendered note has a delete button that will remove it from the database and from the HTML
